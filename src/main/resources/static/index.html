About StegoSecure

        
StegoSecure is a robust platform for secure data hiding, designed for professional and educational use cases where data integrity and confidentiality are paramount. It leverages a combination of modern cryptographic standards and steganographic techniques to invisibly embed data within images.

        
Core Architecture

        
The application is built with a backend-first approach, ensuring a stable and secure foundation. The data flows through a clean, modular architecture:

        
    
            
  • A JavaScript Frontend provides a clean user interface for interacting with the system.
    • 
            
  • A Java Spring Boot Backend handles all core logic via a RESTful API. The StegoController manages HTTP requests.
    • 
            
  • The StegoService layer orchestrates the business logic, coordinating between encryption and embedding processes.
    • 
            
  • Utility Classes (AESUtil and ImageSteganographyUtil) handle the low-level operations of AES encryption and Least Significant Bit (LSB) steganography.
    • 
        

        
Security Principles

        
Security is the primary focus. Messages are first encrypted using AES, and only the resulting ciphertext is embedded. This ensures that even if the presence of a hidden message is detected, its content remains unreadable without the correct key.

    






    

        
        
User Manual

        
How to Encode (Hide) a Message

        
    
            
  1. Select "Encode Message" from the main screen.
    2. 
            
  2. Upload Carrier Image: Click the designated area to select a PNG file from your computer, or simply drag and drop it. A preview will appear.
    3. 
            
  3. Enter Secret Message: Type or paste the text you wish to hide into the message field.
    4. 
            
  4. Set Encryption Key: Provide a strong, private key. This key is critical and must be remembered to retrieve the message later.
    5. 
            
  5. Encode & Download: Click the button. The system will encrypt your message, embed it into the image, and automatically download the new steganographic image (encoded_image.png) to your computer.
    6. 
        


        
How to Decode (Reveal) a Message

        
    
            
  1. Select "Decode Message" from the main screen.
    2. 
            
  2. Upload Steganographic Image: Upload the PNG file that contains the hidden message.
    3. 
            
  3. Enter Decryption Key: You must enter the exact same key that was used during the encoding process.
    4. 
            
  4. Decode Message: Click the button. If the key is correct and a message exists, it will be decrypted and displayed in the results box below.
